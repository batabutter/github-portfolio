Reflective Essay:

My ultimate goal as a Computer Science and Mathematics student is to utilize my knowledge of algorithms and low-level programming to build a foundation to learn other systems. My most proficient programming languages are C and C++; however, I am also proficient in TypeScript and Python.

Currently, I am working on a 6502 emulator. The 6502 is an older CPU that was used in the Commodore64 and the Nintendo Entertainment System. I am emulating the CPU using C++, and so far, I have fully implemented LDA instructions, as well as reading, fetching, and operating on bytes. This project, which is still ongoing, taught me a lot about software design and structures using a modern language like C++. Because I am simply emulating a CPU, there are many design and OOP decisions that I need to make myself when transferring machine instructions to a modern language.

Some of the higher-level projects I have worked on include an environmental sensor monitoring web application for the Department of Environmental Science at the University of Maryland. As a group, we rebuilt the framework for the original sensor monitoring website and continue to make contributions, transferring the systems from the previous site to the new system. The largest difficulty surrounding this project was managing a codebase of which I was unfamiliar. Currently, a small group and I are working on transferring the data to a new repository on GitHub and managing access to the server that holds thousands of sensor readings.

Another project I’m quite passionate about is my Multiplatform Tomodoro Timer. Using ElectronJS, I created a Windows and Linux application that functions as a customizable Pomodoro timer. A pomodoro timer is used to create bite-sized work and break timers that help you stay productive. Currently, you can set various timer presets to separate working times and break times. I am currently working on configuring presets that allow you to customize the colors and visual appearance of the application.

I am a dedicated student and capable programmer, and I am excited to learn more about software design throughout my career. I hope to bring my strong work ethic and design principles to your company.
